# AI Code Documentation Assistant

## Description
An AI-powered assistant that uses Retrieval-Augmented Generation (RAG) to analyze source code, answer developer questions, and automatically generate documentation.

## Features
- Upload multiple code files
- AI-based code explanation
- Automatic documentation generation
- Context-aware answers using RAG
- Supports Python, C, C++, Java and JavaScript

## Technologies Used
- Python
- Gradio
- Google Gemini API
- LangChain
- FAISS
